animation-duration

语法

animation-duration: <time>[,<time>]*; 指定对象动画的持续时间 。
语法项目 说明
初始值 0
适用于 所有元素
可否继承 否
单位 S
媒介 视觉
版本 CSS3.0

说明

检索或设置对象动画的持续时间

如果提供多个属性值，以逗号进行分隔。

取值

<time>：正数，单位可以是秒(s)或者毫秒(ms)。默认值为0，表明动画不执行。

实例代码

CSS 复制代码
 
  1. .animation_duration{
  2.     left:10px;
  3.     top:100px;
  4.     position:absolute;
  5.     -webkit-animation:0.5s ease 1 forwards;
  6.     -webkit-animation-duration:2s;
  7.     -webkit-animation-name:demo;
  8.     -moz-animation:0.5s ease 1 forwards;
  9.     -moz-animation-name:demo;
  10.     -moz-animation-duration:2s;
  11. }
  12. @-webkit-keyframes demo{
  13.     0%{left:10px;}
  14.     100%{left:400px;}
  15. }                        
HTML 复制代码
 
  1. <div class="demo_box animation_duration">我一共运行了2秒钟</div>                        
运行 新窗口运行
