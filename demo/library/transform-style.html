transform-style

语法

transform-style: flat | preserve-3d
语法项目 说明
初始值 flat
适用于 块元素和行内元素
可否继承 否
媒介 视觉
版本 CSS3.0

说明

设置内嵌的元素在 3D 空间如何呈现。有两个值：flat：所有子元素在 2D 平面呈现。preserve-3d：保留3D空间。

取值

flat：所有子元素在 2D 平面呈现。

preserve-3d：保留3D空间。
